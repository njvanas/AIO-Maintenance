# 🛠️ AIO-Maintenance Build Instructions

## Step-by-step to Build the Executable

1. 🧰 Install Python 3.x (recommended 3.9+)
2. 📦 Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. 🏗️ Build the .exe:
   - Run `build.bat` in a terminal or double-click it
   - Output will appear in the `dist/` folder

> This includes all required hidden imports for `requests` and `customtkinter`
